# Inventory

A way to keep track of items in various locations.

I'm building this as a way for me to keep track of things around the house.

## Ideas / Scratchpad

- items can be found by name.
- Store household items in a location.
- items can be moved.
- items can be used.
- items can have a use-by date (perishable).
- items can be discarded.
 - items could be tagged and found by tags.
